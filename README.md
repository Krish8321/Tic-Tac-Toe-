# Tic-Tac-Toe-
A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. <br>
<br>
📌<b>Features </b> <br>
-> Two-player gameplay 👥 <br>
-> Win and draw detection <br>
-> Interactive UI <br>
-> Simple and clean design <br> <br>
🎯 <b>How It Works </b> <br>
-> Two players take turns (X and O) <br>
-> Click on a box to make a move <br>
-> Game checks for winner after each move <br>
-> Displays winner or draw result <br>
