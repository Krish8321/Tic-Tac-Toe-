# Tic-Tac-Toe-
A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. <br>
<br>
📌 Features
Two-player gameplay 👥
Win and draw detection
Interactive UI
Simple and clean design
🎯 How It Works
Two players take turns (X and O)
Click on a box to make a move
Game checks for winner after each move
Displays winner or draw result
